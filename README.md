# SebastianVillalobosAlva.github.io

Personal portfolio site — CV + project write-ups, served as a single static `index.html`.

## Deploy (one time)

1. Create a **new public repo on GitHub named exactly** `SebastianVillalobosAlva.github.io`.
2. From this folder, push it:

   ```bash
   git init
   git add .
   git commit -m "Initial portfolio site"
   git branch -M main
   git remote add origin https://github.com/SebastianVillalobosAlva/SebastianVillalobosAlva.github.io.git
   git push -u origin main
   ```

3. On GitHub: **Settings → Pages → Build and deployment → Source: Deploy from a branch → `main` / `/ (root)`.**
4. Your site goes live at **https://sebastianvillalobosalva.github.io** within a minute or two.

## Edit

Everything lives in `index.html` (HTML + CSS in one file). Open it in a browser to preview locally, edit, and push again to update.
