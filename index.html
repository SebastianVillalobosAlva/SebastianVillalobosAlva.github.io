Research focus

The Mechanistic Structure of Safety in Neural Networks

The thesis

Today's AI safety is behavioral. RLHF and red-teaming teach a model what gets penalized — not why something is harmful. That's why safety breaks under temperature changes, adversarial prompts, and distribution shift: there's no underlying principle, just pattern-matching on what got rewarded.

I'm investigating whether Stoic ethics — the dichotomy of control, temperance, virtue — can be encoded at the representation level to produce safety that is mechanistically grounded, then using interpretability to verify it actually works inside the model.

Two projects, one bridge: a from-scratch interpretability toolkit (ModelLens) and a philosophical steering pipeline (Stoic LLM) — both built on a laptop with a 1B-parameter model.

ModelLens

Architecture-agnostic neural network interpretability toolkit

Point ModelLens at any PyTorch model — transformer, CNN, LSTM, GRU, or MLP — and it auto-detects the architecture and runs the interpretability analyses that make sense for it. Where TransformerLens works only on transformers, ModelLens uses an adapter pattern to give a single unified API across architecture families, declaring which analyses are available and raising a clear error when one doesn't apply.

from modellens import ModelLens lens = ModelLens(model) # auto-detects the architecture results = lens.layer_probe(inputs) # generalized logit lens attn = lens.attention_map(inputs) effects = lens.activation_patch(clean, corrupted) # causal intervention

The analysis suite covers layer probing (generalized logit lens), attention maps, activation patching, residual-stream contribution analysis, embedding inspection, CNN filter & feature-map evolution, LSTM/GRU gate decomposition, and automated circuit discovery — each gated by an AnalysisCapability the adapter declares. Hook lifecycles are managed by context managers, so captured activations are always released even when an analysis throws.

Flagship result. I used ModelLens to run a behavior-controlled circuit comparison on a 3B model: two steering methods (activation addition vs. LoRA fine-tuning) produce the same Stoic-flavored output through different circuits — activation steering concentrates in a single L8→L9–13 feed-forward pathway (layer-12 MLP is the causal bottleneck), while LoRA distributes its effect across many layers and uses attention as well as MLPs. That "same behavior, different mechanism" finding is exactly the kind of question interpretability tooling exists to answer.

Stoic LLM Steering

Philosophical steering toward mechanistically grounded safety

Steers Llama-3.2 (1B → 3B) toward Stoic philosophy — Epictetus, Seneca, Marcus Aurelius — using Contrastive Activation Addition (CAA) and LoRA fine-tuning. The full pipeline is built from scratch: scraping the source texts, generating contrastive Stoic/neutral pairs (100+ per philosopher) via the Claude API, extracting steering vectors, and scoring outputs with an LLM-as-judge across layer × coefficient sweeps.

The motivating idea is philosophy as mechanistic safety — using Stoic principles (the dichotomy of control, temperance, virtue ethics) as a constraint framework rather than an RLHF-style reward signal. But the work's real value is its rigor about what does and doesn't hold:

  • A register direction, not a decision direction. Three independent instruments converge on the same boundary: steering robustly shifts style (+1.0 to +1.6 by LLM-judge), weakly and inconsistently shifts reasoning in prose, and — on a judge-free, generation-free forced-choice dilemma probe — does not move the model's actual choice at any strength, right up to the edge of incoherence. It changes how the model talks, not what it decides.
  • CAA vs. LoRA, mechanistically. Both produce Stoic-flavored text, but ModelLens shows they do so through different circuits — the publishable anchor, and the basis for the open thesis test: does LoRA move decisions where activation steering can't?
  • Built to catch its own artifacts. Seed-averaged, decoding-matched, label-order-balanced probes; capacity isolated as the bottleneck (effects strengthen from 1B to 3B); and an early "all philosophers converge on layer 8" result traced to an extraction bug and explicitly superseded rather than reported.

The result is a clean characterization of what activation steering can and can't do — a more defensible claim than any single positive number, and a sharper question for the next experiment than the project started with.
